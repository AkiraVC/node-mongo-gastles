# Gastles NodeJS en MongoDB

## Chapter 0 - NodeJS warm up

### Terminal

```
$ node
> 1 + 1 
2
> console.log('Hello world')
Hello world
```
> To exit, press ctr+c twice

### Run node script

1. Create a file `script.js`
2. Add some javascript  
```
console.log('Hello world')
```
3. Run the script
```
$ cd path/to/chapter-0/
$ node script.js
> Hello world
```
